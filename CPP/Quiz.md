# C++ Basics Quiz

This quiz is meant to reinforce your understanding of the material covered in the lesson.

---

## Question 1

**What is a statement?**

**Solution:**

A statement is an instruction in a computer program that tells the computer to perform an action.

---

## Question 2

**What is a function?**

**Solution:**

A function is a collection of statements that executes sequentially.

---

## Question 3

**What is the name of the function that all programs must have?**

**Solution:**

`main`

---

## Question 4

**What happens when the program is run?**

**Solution:**

The statements inside `main()` are executed in sequential order.

---

## Question 5

**What symbol are statements in C++ often ended with?**

**Solution:**

A semicolon:

```cpp
;
```

---

## Question 6

**What is a syntax error?**

**Solution:**

A syntax error occurs when your program violates the grammar rules of the C++ language.

---

## Question 7

**What is the C++ Standard Library?**

**Hint:** Review lesson 0.5 — Introduction to the compiler, linker, and libraries.

**Solution:**

The C++ Standard Library is a collection of prewritten code that comes with C++. It provides useful functionality such as input/output, strings, containers, algorithms, math utilities, and more.

Examples include:

```cpp
#include <iostream>
#include <string>
#include <vector>
```
