**Syntax** is set of rules that describe how a set of words (and punctuation) can be arranged to form valid sentences in a language. A **syntax** error is a type of error that get raised by the compiler when your program doesn't follow the language rules of C++
## Statement

**Statement** is a type of instruction that tells the program what to do. They are the smallest independent unit of computation in the C++ language. It is a complete instruction that tells the program what to do.

Expression: produces a value
Statement: performs an action

Expression:
```
x + 1
```

Statement:
```
x = x + 1
```

### Statement types

**Declaration statement**
```
int x = 5;
```
**Expression statement**
```
x = x + 1
```
**Compound statement**
```
{
	int y = 10;
	std::cout << y << std::endl;
}
```
**Selection statement (conditionals)**
```
if (x > 5){
	std::cout << "Valid" << std::endl;
} else {
	std::cout << "Invalid" << std::endl;
}
```
**Iteration statement (loops)**
```
for (int i = 0; i < 10; ++i){
	std::cout << i << std::endl;
}
```
**Try blocks**
```
try {
	throw std::runtime_error("Some Error");
} catch (const std::exception& e) {
	std::cout << e.what() << std::endl;
}
```
**Jump statement**
```
return 0;   // exits a function
break;      // exits a loop or switch
continue;   // skips to next loop iteration
goto label; // jumps to a label
```

## Functions

A **function** is a collection of statements that get executed sequentially. In C++, statements are typically grouped into units called functions. They are typically written to do a specific job or perform some useful action.

The name of a function, object, type, template, etc.... is called its **identifier**.
